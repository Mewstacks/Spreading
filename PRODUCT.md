# Product

## Register

product

## Users

Base mista de afiliados brasileiros (pt-BR):
- **Iniciante leigo**: administra grupos de ofertas no WhatsApp/Telegram, não-técnico, usa muito o celular. Quer conectar contas e ver comissão entrar sem entender de "scraper", "token" ou "id de grupo".
- **Semi-profissional**: opera vários grupos/canais, entende afiliação ML/Amazon, usa desktop, quer densidade de dados, regras finas de envio e eficiência.

Job to be done: publicar automaticamente as melhores ofertas (Mercado Livre + Amazon) nos grupos, com link de afiliado próprio, e acompanhar cliques/comissão.

## Product Purpose

Spreading é um painel SaaS que raspa ofertas dos marketplaces, gera links de afiliado por usuário, envia mensagens automatizadas para WhatsApp/Telegram segundo regras (nicho → canal → grupo → intervalo) e sincroniza receita/comissão de volta. Sucesso = conta operacional em minutos (onboarding "Começar") e receita recorrente visível no dashboard.

## Brand Personality

Confiável, direto, simples. Ferramenta séria que "só funciona". Tom pt-BR coloquial-profissional, sem enfeite; explica termos técnicos inline quando inevitáveis (token, id do grupo).

## Anti-references

- Painel "hacker/scraper" cru: logs crus como interface primária para usuário leigo.
- SaaS genérico de dashboard cheio de widgets sem ação.
- Excesso de jargão de afiliação/tecnologia sem explicação.

## Design Principles

1. **Estado das conexões sempre visível** — ML, WhatsApp, Telegram são o coração; status e caminho de reparo a um clique.
2. **Leigo nunca digita o que a máquina sabe** — ids de grupo, tokens e tags entram por seleção/validação guiada, não por campo cru.
3. **Uma regra, uma linha** — configuração densa para o pro, mas com defaults seguros para o iniciante.
4. **Feedback em tempo real honesto** — envio/scrape mostram progresso e erro com próximo passo, não stack trace.
5. **Mobile-first de verdade** — tudo operável com o polegar; tabelas viram cards.

## Accessibility & Inclusion

WCAG AA básico: contraste 4.5:1 em texto, foco visível, navegação por teclado nos fluxos principais (login, criar regra, enviar oferta), ícones sempre acompanhados de rótulo em navegação.
